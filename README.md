# FIMA - Fondazione Italiana Musica Antica

Official website for the Italian Foundation for Early Music (Fondazione Italiana Musica Antica).

## Tech Stack

- **Framework**: Symfony 8.0
- **PHP**: 8.4+
- **ORM**: Doctrine
- **Template Engine**: Twig

## Prerequisites

- PHP 8.4 or higher
- Composer
- Symfony CLI (recommended)

## Installation

```bash
# Install dependencies
composer install

# Run database migrations
php bin/console doctrine:migrations:migrate

# Start development server
symfony server:start
```

## Development

Access the application at `https://localhost:8000` after starting the server.
