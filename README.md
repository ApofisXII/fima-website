# FIMA - Fondazione Italiana Musica Antica

Sito web ufficiale della Fondazione Italiana Musica Antica.

## Stack

- Symfony 8.0
- PHP 8.4+
- Doctrine ORM
- Twig

## Installazione

```bash
composer install
php bin/console doctrine:migrations:migrate
symfony server:start
```
